### 🚀 AttendEase – AI-Powered Smart Attendance System

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![OpenCV](https://img.shields.io/badge/OpenCV-5C3EE8?style=flat-square&logo=opencv&logoColor=white)
![AI](https://img.shields.io/badge/AI-Face_Recognition-orange?style=flat-square)

An AI-based attendance management system that automates attendance using facial recognition and liveness detection. The system prevents proxy attendance, supports multi-face detection, and syncs records to the cloud in real time.

**🔹 Features**
- Real-time face recognition
- Anti-spoofing security
- Multi-face attendance marking
- Cloud synchronization
- Attendance analytics dashboard

**🏆 Highlights**
- 99.5% recognition accuracy
- Attendance marked in under 5 seconds
- Saves 10+ minutes per class

### 🔥 Ignisia – Real-Time Crowd Risk Detection & Stampede Prevention

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![YOLOv8](https://img.shields.io/badge/YOLOv8-Computer_Vision-red?style=flat-square)
![OpenCV](https://img.shields.io/badge/OpenCV-5C3EE8?style=flat-square&logo=opencv&logoColor=white)

An AI-powered surveillance solution that analyzes live CCTV feeds to detect crowd congestion and potential stampede risks. It uses computer vision and density analysis to generate real-time alerts for proactive crowd management.

**🔹 Features**
- YOLOv8-based person detection
- Crowd density monitoring
- Automated risk alerts
- Interactive dashboard
- Real-time analytics & heatmaps

**🏆 Highlights**
- Automated crowd risk detection
- Reduced dependence on manual monitoring
- Scalable multi-camera architecture
